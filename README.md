# 📚 Sobre o curso

O **Java Spring Expert** é uma formação da **DevSuperior**, ministrada
pelo professor **Nélio Alves**, voltada ao desenvolvimento de APIs REST
profissionais utilizando Java e Spring Boot.

🔗 **Curso oficial:**
https://devsuperior.com.br/curso-java-spring-expert


------------------------------------------------------------------------

# 🎯 Objetivo

Este repositório reúne os estudos, exercícios e desafios desenvolvidos
durante a formação **Java Spring Expert**, abrangendo desenvolvimento de
APIs REST, segurança, testes automatizados, cobertura de código e testes
de APIs utilizando o ecossistema Spring.
